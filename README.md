# Proyecto Dato Global AU

Explorador web de actividad sísmica mundial basado en datos públicos del USGS.

Permite revisar eventos recientes, filtrar por país y magnitud, consultar información sobre Chile y aprender cómo prepararse ante un terremoto.

## Funciones principales

- Datos sísmicos mundiales actualizados automáticamente.
- Filtros por país, ubicación y magnitud.
- Estadísticas, gráficos y seguimiento de eventos en Chile.
- Modo claro y oscuro.
- Guía de preparación, mochila de emergencia y plan familiar.

## Tecnologías utilizadas

- HTML5
- JavaScript
- Tailwind CSS
- Chart.js
- API pública del USGS Earthquake Hazards Program

## Página publicada

https://franciscotravisany.github.io/proyectodatoglobalau/

## Cómo abrirlo localmente

Descarga o clona el repositorio y abre `index.html` mediante un servidor local, como la extensión Live Server de Visual Studio Code. El proyecto no necesita instalación ni claves de API.

## Fuente de los datos

La información sísmica utilizada por el explorador proviene del [USGS Earthquake Hazards Program](https://earthquake.usgs.gov/).
